Xuanpu Workbench

An AI-native workbench for builders.
Projects, context, agents, changes — one workspace.

One workspace for everything

Projects, agent sessions, file trees, diffs, and git state — all in a single window. No more scattering context across terminal tabs, editors, and browser windows.

Isolated branches by default

Every task runs on its own git worktree branch. Multiple agents can work on the same repo simultaneously without conflicts, no stashing, no coordination overhead.

Cross-repo context with Connections

Link repositories together so a single agent session sees your entire stack. Frontend, backend, shared libraries — one agent reasons and implements across all of them in one pass.
